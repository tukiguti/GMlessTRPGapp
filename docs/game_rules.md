## 概要

このドキュメントは、GMレスLoL風TRPGのゲームルールの総合索引です。
ゲームルールは以下の4つのドキュメントに分割されています。

### 2. [キャラクターシステム](./character_system.md)

キャラクターのステータス、レベルシステム、スキルシステムなど。

**含まれる内容**:
- ステータスシステム
  - 基本ステータス（HP、攻撃力、防御力、移動力、ユーティリティ）
- ロール別初期ステータス
  - ADロール（マークスマン、ファイター、アサシン、タンク）
  - APロール（メイジ、アサシン、ファイター、タンク）
  - サポートロール（APサポート、タンクサポート）
- レベルアップシステム
  - レベル上限（最大18）
  - 経験値獲得方法
  - 必要経験値テーブル
  - レベルアップ時の成長率
  - レベル18時の最終ステータス
  - ロール特性
- スキルシステム
  - スキル構成（通常スキル1個 + ULT1個）
  - クールダウンシステム（通常2ターン、ULT5ターン）
  - スキル効果の種類
    1. ダメージ系スキル（攻撃強化）
    2. CC系スキル（クラウドコントロール）
    3. バフ系スキル（味方強化）
    4. デバフ系スキル（敵弱体化）

  - スキル使用の戦略例

### 3. [戦闘・判定システム](./combat_system.md)

視界システム、マッチアップ判定、ダメージ計算、デスペナルティ、複数人戦闘など。

**含まれる内容**:
- 視界システム
  - ワード配置（配置可能エリア、配置数制限）
  - ワード破壊（判定方法）
  - 視界の効果（視界がある場合/ない場合）
- 基本戦闘
  - マッチアップ判定
    - 判定手順（ダイスロール、ステータス補正、視界補正、合計値比較）
    - 判定式: `1d20 + 攻撃力補正 + 状況補正`
    - 判定例
- ダメージ計算
  - ダメージ計算式: `(差分) × 10 - 防御力`
  - ダメージ計算例
- デスとペナルティ
  - デス条件（HPが0以下）
  - デスペナルティ
    - リスポーン時間: `1ターン + (レベル × 0.2ターン)`
    - ゴールド喪失（10%）
    - 経験値提供（+0.5経験値ターン）
    - リスポーン位置（自陣ネクサス）
  - 復活後の状態
- **複数人戦闘**（2025-11-10 詳細化）
  - 基本ルール（戦闘参加条件、戦闘処理の原則）
  - ターゲット選択ルール
    - ターゲット宣言タイミング（同時宣言・同時公開）
    - ターゲット選択の制約（視界内、同一エリア、ポジション制約）
    - ターゲット変更ルール
  - **ポジショニングシステム**
    - フロントライン（タンク、ファイター）
    - バックライン（マークスマン、メイジ、サポート）
    - フレキシブル（アサシン）
    - ポジション制約によるターゲット制限
    - アサシンのバックライン突入（1d20 + 移動力 ≧ 14）
  - 数的有利とボーナス
    - 2v1（+2）、3v1（+3）、4v1（+4）、5v1（+5）
    - 不均衡な戦闘（3v2、4v2など）
    - フォーカスターゲット
  - 戦闘解決の流れ（6ステップ）
  - AOEスキルの処理
  - 複数人戦闘の例（2v1、3v3、アサシン突入、AOEスキル）
- 戦闘時の補正まとめ

  - ハイブリッド系: トリニティフォース

## クイックリファレンス

### 基本的なゲームの流れ
1. **ゲーム開始**: チーム編成、ロール選択 → [ゲーム概要](./game_overview.md)
2. **キャラクター作成**: ステータス、スキル選択 → [キャラクターシステム](./character_system.md)
3. **ターン進行**: 配置 → 行動宣言 → 行動解決 → 結果処理 → [ゲーム概要](./game_overview.md)
4. **戦闘**: マッチアップ判定、ダメージ計算 → [戦闘・判定システム](./combat_system.md)
5. **ガンク/オブジェクト**: 視界管理、ガンク判定、オブジェクト挑戦 → [ゲームプレイシステム](./gameplay_systems.md)
6. **成長**: 経験値獲得、レベルアップ → [キャラクターシステム](./character_system.md)
7. **アイテム購入**: リコール後またはリスポーン時にアイテム購入・合成 → [アイテムシステム](./item_system.md)
8. **勝利**: ネクサス破壊 → [ゲーム概要](./game_overview.md)

### よく使う判定

| 判定 | ダイス | 補正 | 成功条件 | 参照 |
|------|--------|------|---------|------|

| **CC判定** | 1d20 | +ユーティリティ | 13以上 | [ゲームプレイシステム](./gameplay_systems.md) |

| **ワード破壊判定** | 1d20 | なし | 12以上 | [戦闘・判定システム](./combat_system.md) |
