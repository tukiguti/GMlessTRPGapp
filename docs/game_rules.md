# GMレスLoL風TRPG ゲームルール 総合索引

このドキュメントは、GMレスLoL風TRPGのゲームルールの総合索引です。
ゲームルールは以下の4つのドキュメントに分割されています。

## ドキュメント構成

### 1. [ゲーム概要とゲームシステム](./game_overview.md)

ゲームの基本的なコンセプト、マップ構造、ターン構造、勝利条件など。

**含まれる内容**:
- ゲーム概要
  - 基本コンセプト（GMレス、ダイスベース、5vs5チーム戦）
- 基本ルール
  - チーム構成
  - レーンとロールの区別
  - レーン（TOP/JG/MID/BOT）
  - ロール（Tank/Fighter/Assassin/Mage/Marksman/Support）
- マップ構造
  - タワー配置（ネクサス、ネクサスタワー、インナー、アウター）
  - エリア分類（レーン、ジャングル、中立エリア）
  - バフ配置（赤バフ、青バフ）
- ターン構造
  - 配置フェーズ
  - 行動宣言フェーズ
  - 行動解決フェーズ
  - 結果処理フェーズ
- 勝利条件

### 2. [キャラクターシステム](./character_system.md)

キャラクターのステータス、レベルシステム、スキルシステムなど。

**含まれる内容**:
- ステータスシステム
  - 基本ステータス（HP、攻撃力、防御力、移動力、ユーティリティ）
- ロール別初期ステータス
  - ADロール（マークスマン、ファイター、アサシン、タンク）
  - APロール（メイジ、アサシン、ファイター、タンク）
  - サポートロール（APサポート、タンクサポート）
- レベルアップシステム
  - レベル上限（最大18）
  - 経験値獲得方法
  - 必要経験値テーブル
  - レベルアップ時の成長率
  - レベル18時の最終ステータス
  - ロール特性
- スキルシステム
  - スキル構成（通常スキル1個 + ULT1個）
  - クールダウンシステム（通常2ターン、ULT5ターン）
  - スキル効果の種類
    1. ダメージ系スキル（攻撃強化）
    2. CC系スキル（クラウドコントロール）
    3. バフ系スキル（味方強化）
    4. デバフ系スキル（敵弱体化）
    5. 移動系スキル
    6. 範囲バフ系スキル
  - スキル使用の制約
  - スキル使用の戦略例

### 3. [戦闘・判定システム](./combat_system.md)

視界システム、マッチアップ判定、ダメージ計算、デスペナルティなど。

**含まれる内容**:
- 視界システム
  - ワード配置（配置可能エリア、配置数制限）
  - ワード破壊（判定方法）
  - 視界の効果（視界がある場合/ない場合）
- 基本戦闘
  - マッチアップ判定
    - 判定手順（ダイスロール、ステータス補正、視界補正、合計値比較）
    - 判定式: `1d20 + 攻撃力補正 + 状況補正`
    - 判定例
- ダメージ計算
  - ダメージ計算式: `(差分) × 10 - 防御力`
  - ダメージ計算例
- デスとペナルティ
  - デス条件（HPが0以下）
  - デスペナルティ
    - リスポーン時間: `10秒 + (レベル × 2秒)`
    - ゴールド喪失（10%）
    - 経験値提供（キラー50XP、アシスト25XP）
    - リスポーン位置（自陣ネクサス）
  - 復活後の状態
- 複数人戦闘
  - 攻撃対象選択
  - 数的有利（2v1など: +2ボーナス）
  - 複数人戦闘の例
- 戦闘時の補正まとめ

  - タワーダメージ計算
  - タワー破壊の順序制約
  - CSとタワー攻撃の関係まとめ

  - オブジェクト種類（ドラゴン、バロン、ヘラルド）
  - オブジェクト挑戦の流れ
  - 戦闘とオブジェクトの同時発生

  - ファームシステム

  - ゴールド・経験値

## クイックリファレンス

### 基本的なゲームの流れ
1. **ゲーム開始**: チーム編成、ロール選択 → [ゲーム概要](./game_overview.md)
2. **キャラクター作成**: ステータス、スキル選択 → [キャラクターシステム](./character_system.md)
3. **ターン進行**: 配置 → 行動宣言 → 行動解決 → 結果処理 → [ゲーム概要](./game_overview.md)
4. **戦闘**: マッチアップ判定、ダメージ計算 → [戦闘・判定システム](./combat_system.md)
5. **ガンク/オブジェクト**: 視界管理、ガンク判定、オブジェクト挑戦 → [ゲームプレイシステム](./gameplay_systems.md)
6. **成長**: 経験値獲得、レベルアップ → [キャラクターシステム](./character_system.md)
7. **勝利**: ネクサス破壊 → [ゲーム概要](./game_overview.md)

### よく使う判定

| 判定 | ダイス | 補正 | 成功条件 | 参照 |
|------|--------|------|---------|------|
| **マッチアップ判定** | 1d20 | +攻撃力+状況補正 | 高い方が勝利 | [戦闘・判定システム](./combat_system.md) |
| **CC判定** | 1d20 | +ユーティリティ | 13以上 | [ゲームプレイシステム](./gameplay_systems.md) |
| **CC回避判定** | 1d20 | +移動力 | 15以上 | [ゲームプレイシステム](./gameplay_systems.md) |
| **ワード破壊判定** | 1d20 | なし | 12以上 | [戦闘・判定システム](./combat_system.md) |
