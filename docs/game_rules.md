## ゲーム概要

LoL（League of Legends）をベースにしたGMレスTRPG。
ダイスロールとマッチアップ判定を用いて、戦略的な5vs5チーム戦を再現する。

### 基本コンセプト
- **GMレス**: ゲームマスター不要で自動進行
- **ダイスベース**: ランダム性と戦略性のバランス
- **5vs5チーム戦**: ロール分担による協力プレイ
- **戦略性重視**: 視界管理、マッチアップ、リソース配分

## 基本ルール

### チーム構成
- **プレイヤー数**: 5vs5（計10人）
- **チーム**: ブルーチーム vs レッドチーム

### ロール（役割）
各プレイヤーは以下のいずれかのロールを担当：

1. **TOP（トップレーナー）**
   - 担当レーン: TOPレーン
   - 特徴: 1v1の戦闘力重視

2. **JG（ジャングラー）**
   - 担当: 森エリア
   - 特徴: 機動力、ガンク能力
   - 特殊能力: バフファーム、ガンク時の視界ボーナス

3. **MID（ミッドレーナー）**
   - 担当レーン: MIDレーン
   - 特徴: 高いバースト火力、ローム能力

4. **ADC（アタックダメージキャリー）**
   - 担当レーン: BOTレーン
   - 特徴: 後半火力特化、序盤は弱い

5. **SUP（サポート）**
   - 担当レーン: BOTレーン（ADCと同行）
   - 特徴: 視界管理、ガンク能力
   - 特殊能力: ワード多数配置、ガンク時の視界ボーナス

### マップ構造

#### タワー配置（各チーム）
各チームは以下の構造を持つ：

**ブルーチーム（左下）/ レッドチーム（右上）**

マップは3つのレーンで構成され、レーン間にジャングル（森）が配置される：

```
                        [敵ネクサス]
                             |
                      [敵ネクサスタワー]
                             |
              +--------------+--------------+
              |              |              |
         [TOPインナー]  [MIDインナー]  [BOTインナー]
              |              |              |
         [TOPアウター]       |        [BOTアウター]
              |              |              |
              |         [MIDアウター]       |
              |              |              |
        +-----+              |              +-----+
        |  敵側TOP-MID森     |      敵側MID-BOT森  |
        |  (赤バフ/青バフ)   |     (赤バフ/青バフ)  |
        +-----+              |              +-----+
              |              |              |
              +------[中立エリア]-------+
                     (ドラゴン/バロン)
              |              |              |
        +-----+              |              +-----+
        | 味方TOP-MID森      |      味方MID-BOT森   |
        | (赤バフ/青バフ)    |     (赤バフ/青バフ)  |
        +-----+              |              +-----+
              |              |              |
              |         [MIDアウター]       |
              |              |              |
         [TOPアウター]       |        [BOTアウター]
              |              |              |
         [TOPインナー]  [MIDインナー]  [BOTインナー]
              |              |              |
              +--------------+--------------+
                             |
                     [味方ネクサスタワー]
                             |
                        [味方ネクサス]
```

#### タワー構成（1チームあたり）
- **ネクサス**: 1個（最終目標）
- **ネクサスタワー**: 1個（ネクサスを守護）
- **レーンインナータワー**: 3個（TOP/MID/BOT）
- **レーンアウタータワー**: 3個（TOP/MID/BOT）
- **合計**: ネクサス1個 + タワー7個

#### 進行ルート
タワーは以下の順序で破壊する必要がある：
1. **レーンアウタータワー**（各レーン）
2. **レーンインナータワー**（各レーン）
3. **ネクサスタワー**
4. **ネクサス**（破壊で勝利）

**制約**: インナータワーを攻撃するには、そのレーンのアウタータワーを先に破壊する必要がある

#### エリア分類

**レーンエリア**（3レーン × 2チーム = 6エリア）
- **TOPレーン**: 各チームにアウター・インナータワー
- **MIDレーン**: 各チームにアウター・インナータワー
- **BOTレーン**: 各チームにアウター・インナータワー

**ジャングルエリア**（レーン間の森）
- **TOP-MID間ジャングル**:
  - 味方側TOP-MIDジャングル（赤バフまたは青バフ）
  - 敵側TOP-MIDジャングル（赤バフまたは青バフ）
- **MID-BOT間ジャングル**:
  - 味方側MID-BOTジャングル（赤バフまたは青バフ）
  - 敵側MID-BOTジャングル（赤バフまたは青バフ）

**バフ配置**（チームごと）
- **赤バフ**: TOP-MIDジャングルまたはMID-BOTジャングル
- **青バフ**: TOP-MIDジャングルまたはMID-BOTジャングル
  - 通常は対角に配置（例：味方赤バフがTOP-MID側なら、味方青バフはMID-BOT側）

**中立エリア**（マップ中央）
- **ドラゴンピット**: BOT側
- **バロンピット**: TOP側
- **ヘラルドエリア**: TOP側（バロンの前段階）

## ターン構造

### フェーズ制
ゲームは複数のフェーズで構成される。

#### 基本フェーズ順序
1. **配置フェーズ**（ターン開始時）
2. **行動宣言フェーズ**（同時非公開）
3. **行動解決フェーズ**（同時実行）
4. **結果処理フェーズ**

### フェーズの詳細

#### 1. 配置フェーズ
- プレイヤーは自分の位置を選択
- ワード配置（SUPは複数可能）
- リコール（帰還）の実行

#### 2. 行動宣言フェーズ
各プレイヤーは以下から1つを選択（非公開）：
- **レーン戦**: マッチアップ判定を行う
- **ファーム**: リソース獲得（JGはバフ選択可能）
- **ガンク**: 他レーンへ移動して戦闘支援
- **オブジェクト挑戦**: ドラゴン/バロン/ヘラルド
- **ワード破壊**: 敵ワードの破壊試行
- **リコール**: 拠点に戻る

#### 3. 行動解決フェーズ
**重要**: 戦闘フェーズとオブジェクトフェーズは同時に発生する可能性がある

**例**:
- TOPで戦闘発生
- 同時にJGがオブジェクト（ドラゴン）開始
- MIDがローム（移動）

解決順序：
1. 移動系行動の解決
2. 戦闘の発生判定
3. 戦闘解決（複数箇所で同時発生可能）
4. オブジェクト挑戦（戦闘と同時進行可能）

## 視界システム

### ワード（視界提供アイテム）

#### ワード配置
- **配置可能エリア**:
  - **レーン**: TOPレーン、MIDレーン、BOTレーン（各チーム側）
  - **ジャングル**:
    - 味方TOP-MIDジャングル
    - 味方MID-BOTジャングル
    - 敵TOP-MIDジャングル
    - 敵MID-BOTジャングル
  - **中立エリア**: ドラゴンピット、バロンピット付近

- **配置数制限**:
  - **SUP**: 3個まで配置可能
  - **その他のロール**: 1個まで配置可能

#### ワード破壊
- **発動条件**: プレイヤーがワードのあるエリアに進入
- **判定方法**: ダイス判定
  - 破壊判定ダイス: 1d20
  - 成功値: 12以上で破壊成功
- **視認**: ワードが存在することは進入時に自動で判明

### 視界の効果

#### 視界がある場合
- そのエリアに敵が進入したことを即座に把握
- ガンクを事前に察知可能
- マッチアップ判定にボーナスなし（通常判定）

#### 視界がない場合
- 敵の進入を把握不可
- ガンクされた際にペナルティ
- **JGとSUPのガンク時に+判定**（攻撃側有利）

## 戦闘システム

### 基本戦闘（1v1）

#### マッチアップ判定
1. **ダイスロール**: 両者が1d20をロール
2. **ステータス補正**: キャラクターの能力値を加算
3. **視界補正**: 視界状況に応じた補正
4. **合計値比較**: 高い方が勝利

#### ダメージ計算
- 勝利時: `(勝者の出目 - 敗者の出目) × ダメージ係数`
- 引き分け: 両者にダメージなし

### 複数人戦闘（2v2など）

#### 攻撃対象選択
複数人が戦闘に参加する場合：
1. **初回攻撃**: マッチアップ判定実行
2. **継続戦闘の場合**: 再度攻撃対象を選択
   - 各プレイヤーが攻撃対象を指定
   - 同時公開して解決
3. 各ペアごとにマッチアップ判定

#### 数的有利
- 2v1などの場合、攻撃側に有利な補正

## ガンクシステム

### ガンクの流れ

#### 1. ガンク宣言
- JGまたはSUPが他レーンへの移動を宣言
- MIDもローム可能

#### 2. 視界判定
- **視界なし**: ガンク成功、マッチアップ判定に**+3ボーナス**
- **視界あり**: 通常のマッチアップ判定

#### 3. CC（クラウドコントロール）判定
- ガンク側がCC持ちスキルを使用する場合
- **CC判定ダイス**: 1d20
- **成功値**: 13以上でCC成功
- **CC成功時**: 相手のマッチアップ判定に**-5ペナルティ**

#### 4. マッチアップ判定
- 通常の戦闘と同様
- 視界ボーナス、CCペナルティを適用
- 複数人の場合は攻撃対象選択

### ガンク時の補正まとめ

| 状況 | 補正 |
|------|------|
| 視界なし（JG/SUP） | +3 |
| CC成功 | 相手に-5 |
| 数的有利（2v1） | +2 |

## オブジェクトシステム

### オブジェクト種類

#### 1. ドラゴン
- **効果**: チーム全体にバフ付与
- **挑戦難易度**: 中
- **必要ターン**: 2ターン

#### 2. バロン
- **効果**: チーム全体に強力なバフ
- **挑戦難易度**: 高
- **必要ターン**: 3ターン
- **推奨人数**: 2人以上

#### 3. ヘラルド
- **効果**: レーンプッシュ支援
- **挑戦難易度**: 中
- **必要ターン**: 2ターン

### オブジェクト挑戦の流れ

1. **宣言**: オブジェクト挑戦を宣言
2. **判定**: ダイス判定で成功判定
3. **妨害**: 敵が同じタイミングで接近した場合、戦闘発生
4. **獲得**: 成功時、チームにバフ付与

### 戦闘とオブジェクトの同時発生

**例**:
- TOPレーンで戦闘発生
- 同時にJGがドラゴン挑戦開始
- 両方を並行処理

## リソース管理

### ファームシステム

#### JGのバフファーム
- **選択肢**（ジャングルエリア別）:
  - **味方TOP-MIDジャングル**（赤バフまたは青バフ）
  - **味方MID-BOTジャングル**（赤バフまたは青バフ）
  - **敵TOP-MIDジャングル**（赤バフまたは青バフ）
  - **敵MID-BOTジャングル**（赤バフまたは青バフ）

- **ファーム時の位置選択**:
  - JGはどのジャングルエリアでファームするか選択可能
  - 各ジャングルエリアには赤バフまたは青バフが配置されている

- **視界との関係**:
  - 視界がないエリアでファームすることで、敵に位置を悟られない
  - 敵陣バフを狩る「カウンタージャングル」が可能
  - 例：敵MID-BOTジャングルでファームしている場合、敵が視界を持っていなければ位置バレしない

#### バフ効果
- **赤バフ**: 攻撃力上昇、スロー効果
- **青バフ**: クールダウン短縮、マナ回復

### ゴールド・経験値
- **獲得方法**:
  - ファーム成功
  - 戦闘勝利
  - オブジェクト獲得
  - 敵撃破
- **使用用途**:
  - アイテム購入
  - レベルアップ
  - ステータス強化
