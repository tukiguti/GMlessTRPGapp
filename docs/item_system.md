## アイテムシステム概要

### 基本概念
アイテムは以下の3つの階層に分かれています：

1. **基本アイテム（コンポーネント）**: 単体で購入可能な基礎アイテム
2. **上位アイテム（合成）**: 複数の基本アイテムを組み合わせて生成
3. **最上位アイテム**: さらに強力な効果を持つ最終形態

### アイテムスロット
- 各プレイヤーは**最大6個**までアイテムを所持可能
- アイテムスロットが満杯の場合、新規購入・合成不可
- アイテムは売却可能（購入価格の70%で売却）

### アイテムの効果タイプ

#### 1. ステータス効果（パッシブ）
- 所持しているだけで常時ステータスが上昇
- 攻撃力、防御力、HP、移動力、ユーティリティなど

#### 2. 特殊パッシブ効果
- 所持しているだけで常時発動する特殊効果
- クリティカル、ライフスティール、貫通など

#### 3. アクティブ効果
- 使用時に発動する効果
- クールダウンあり（アイテムごとに設定）
- ターン開始時に使用宣言

## 購入とゴールド管理

### 購入タイミング
アイテムは以下のタイミングで購入可能：

1. **リコール成功後**: 自陣ネクサスに戻った時
2. **デス後のリスポーン時**: 自陣ネクサスから復活した時

**重要**: レーン上やジャングル内では購入不可

### ゴールド獲得方法

| 行動 | ゴールド獲得量 |
|------|--------------|
| **ファーム** | 100G |
| **攻撃選択** | 50G |
| **バフファーム（JG）** | 80G + バフ効果 |
| **敵キル** | 300G |
| **敵アシスト** | 150G |
| **タワー破壊** | 500G（チーム全員） |
| **ドラゴン撃破** | 300G（チーム全員） |
| **バロン撃破** | 500G（チーム全員） |

### アイテム合成
- 合成に必要な基本アイテムを全て所持している場合、**追加ゴールド**を支払って合成可能
- 合成後、基本アイテムは消費され、上位アイテムに置き換わる
- 合成はリコール中またはリスポーン時に実行可能

## アイテムカテゴリ

### 1. AD（物理攻撃）系
- 攻撃力を強化
- ADマークスマン、ADファイター、ADアサシンに適している

### 2. AP（魔法攻撃）系
- 魔法攻撃力を強化
- APメイジ、APアサシン、APファイターに適している

### 3. 防御系
- 防御力、HPを強化
- タンク、ファイターに適している

### 4. ユーティリティ系
- 移動力、ユーティリティ、視界、CDR（クールダウン短縮）
- サポート、アサシンに適している

### 5. ハイブリッド系
- 複数のステータスを同時に強化
- 柔軟なビルドが可能

## 特殊効果の詳細

### パッシブ効果の種類

#### 1. クリティカル（インフィニティエッジ）
- **発動条件**: マッチアップ判定のダイスが18以上
- **効果**: ダメージ1.5倍
- **計算例**:
  - 通常ダメージ: (差分6) × 10 - 防御力5 = 55ダメージ
  - クリティカル時: 55 × 1.5 = 82.5ダメージ

#### 2. 防御貫通・防御力減衰（ブラッククリーバー）
- **防御貫通**:
  - **発動条件**: マッチアップ判定で勝利
  - **効果**: 相手の防御力を-2（3ターン持続、スタック可能）
- **防御力減衰**:
  - **効果**: 相手の防御力を20%減衰（常時発動）
  - **計算順序**: 減衰→貫通の順で計算
  - **計算例**:
    - 敵の防御力: 10.0
    - 減衰後: 10.0 × 0.8 = 8.0
    - 貫通適用（1スタック）: 8.0 - 2 = 6.0
    - 最終的な実効防御力: 6.0
- **用途**: 高防御力の敵（タンク）に対して非常に有効

#### 3. 魔力増幅（ラバドンデスキャップ）
- **効果**: 全アイテムから得られる攻撃力を+20%増加
- **計算例**:
  - 基礎攻撃力: +5.0
  - アイテム攻撃力: +8.0
  - ラバドン効果: +8.0 × 0.2 = +1.6
  - 合計: 5.0 + 8.0 + 1.6 = +14.6

#### 3-2. 虚無（ヴォイドスタッフ）
- **効果**: 敵の防御力を40%減衰（常時発動）
- **計算例**:
  - 敵の防御力: 10.0
  - 減衰後: 10.0 × 0.6 = 6.0
  - ダメージ計算: (差分) × 10 - 6.0
- **用途**: 高防御力の敵（タンク）に対して魔法ダメージを大幅に通す

#### 4. 炎上（サンファイアケープ）
- **効果**: 毎ターン開始時、隣接する敵全員に20ダメージ（防御力無視）
- **隣接の定義**: 同じレーン、またはガンク時の敵
- **用途**: 前線で複数の敵に継続ダメージ

#### 5. 復活（ガーディアンエンジェル）
- **効果**: デス時、即座にHP50%で復活
- **制約**: 1回のみ（使用後は通常の防御アイテムとして機能）
- **用途**: 重要な戦闘で生存確率を上げる

#### 6. 高速移動（モビリティブーツ）
- **効果**: ローム/ガンク時、移動力ボーナスが2倍
- **計算例**:
  - 移動力: 5 + 3（ブーツ） = 8
  - ガンクボーナス: (8 - 6) × 2 = 4
- **用途**: ガンク成功率を大幅に向上

#### 7. 味方強化（アーデントセンサー）
- **効果**: 自分のバフスキルを受けた味方に追加で攻撃力+1（2ターン）
- **用途**: サポートがバフスキルと組み合わせて使用

#### 8. 三位一体（トリニティフォース）
- **効果**: スキル使用後、次のマッチアップ判定に+3ボーナス
- **用途**: スキルを頻繁に使用するファイターに最適

### アクティブ効果の種類

#### 1. 停止（ゾーニャの砂時計）
- **使用タイミング**: ターン開始時に宣言
- **効果**: このターン中、無敵状態（攻撃不可、ダメージ受けない）
- **クールダウン**: 5ターン
- **用途**: 敵の強力なスキルやガンクを回避

## アイテム効果の計算

### ステータス計算の優先順序

1. **基礎ステータス**: キャラクターの基本ステータス + レベル成長
2. **アイテム基礎効果**: 各アイテムのステータス加算
3. **パッシブ効果**: ラバドンの魔力増幅など、他の効果を増幅
4. **バフ効果**: スキルやアイテムによる一時的な強化

### 計算例

**前提**:
- レベル10のAPメイジ
- 基礎攻撃力: +9.5
- 所持アイテム:
  1. ラバドンデスキャップ（攻撃力+6.0、魔力増幅+20%）
  2. ゾーニャの砂時計（攻撃力+3.5）
  3. アンプリファイングトーム（攻撃力+1.5）

**計算**:
1. 基礎攻撃力: +9.5
2. アイテム攻撃力合計: +6.0 + 3.5 + 1.5 = +11.0
3. ラバドン魔力増幅: +11.0 × 0.2 = +2.2
4. **最終攻撃力**: 9.5 + 11.0 + 2.2 = **+22.7**

### 複数パッシブの重複

#### 重複可能なパッシブ
- **防御貫通**: スタック可能（最大-10まで）
- **炎上**: 複数所持しても効果は1つのみ

#### 重複不可なパッシブ
- **クリティカル**: 複数のクリティカルアイテムを持っても倍率は1.5倍固定
- **復活**: 複数所持しても1回のみ発動
